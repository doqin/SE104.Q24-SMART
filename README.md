# S-MART: Sales Management System (Mocked)
## Mô tả: 
Hệ thống quản lý bán hàng toàn diện dành cho cửa hàng bán lẻ.

## Công nghệ sử dụng:

### Frontend: 
React 18, TailwindCSS

### Backend: 
Node.js, Express

### Database: 
PostgreSQL

#### Hướng dẫn cài đặt (Local):

1. Clone repository này: git clone https://github.com/your-username/s-mart.git

2. Di chuyển vào thư mục server: cd server và chạy npm install.

3. Cấu hình file .env (Database URI, JWT Secret).

4. Chạy backend: npm run dev.

5. Lặp lại bước 2 và 4 cho thư mục client.
